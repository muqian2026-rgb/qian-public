# domain-learn

> 领域学习（无单一教材）：年表/框架 → 深读节点 → 交互探索 HTML。  
> 与 [book-learn-distill](../book-learn-distill/) 并列，走 `pipeline=domain`。

## 安装

```bash
git clone https://github.com/muqian2026-rgb/qian-public.git
cp -r qian-public/domain-learn ~/.cursor/skills/domain-learn
```

需同时安装 [book-learn-distill](../book-learn-distill/)（共享 Intake 与构建脚本）。

## 口令

```
领域学习 行业演进史
年表学习 某学科时间轴
时间机器 某方向
```

## 模板

复制 [learn/_template-domain/](./learn/_template-domain/) 到你的学习仓库 `learn/<slug>/`，按 SKILL 六阶段推进。
