## 快速开始

```bash
git clone https://github.com/muqian2026-rgb/qian-public.git
cp -r qian-public/book-learn-distill ~/.cursor/skills/book-learn-distill
```

在 Cursor 里说：

```
学一下 行为经济学
```

完整产出在**你的私有库**生成；本仓库提供 Skill 源码。
