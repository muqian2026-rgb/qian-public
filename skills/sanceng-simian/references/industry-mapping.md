# 行业映射示例 · 三层四面

> 通用问句骨架。执行时替换为你的行业指标与数据来源。

## 三层 · 问句

| 层 | 问句 | 数据来源 |
|----|------|----------|
| 大盘 | 全行业交易规模？增速？ | 统计局、研报、行业协会 |
| 线上化率 | 线上占行业比例？ | 渠道调研、竞品公开数据 |
| 市占率 | 我方占线上市场份额？ | 内部 GMV + 外部估算 |

## 四面 · 指标

| 面 | 常见指标 |
|----|----------|
| 用户 | MAU、付费用户、新增用户 |
| 频次 | 人均年消费次、复购率 |
| 客单价 | GMV/订单量、均价 |
| 货币化率 | 抽佣率、毛利率、ARPU |
